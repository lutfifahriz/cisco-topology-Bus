# Bus Topology Simulation

Simulasi jaringan menggunakan topologi Bus pada Cisco Packet Tracer.

## Deskripsi
Project ini menggunakan:
- 1 Server
- 5 Switch
- 5 PC

Server dikonfigurasi sebagai DHCP Server, sedangkan seluruh PC menggunakan DHCP Client agar mendapatkan IP secara otomatis.

## Konfigurasi
- DHCP Server
- Automatic IP Addressing
- Basic Network Connectivity

## Implementasi
Topologi Bus cocok digunakan untuk jaringan sederhana dengan kebutuhan perangkat yang tidak terlalu banyak.

## Kelebihan
- Struktur sederhana
- Mudah dipahami untuk pembelajaran dasar jaringan
- Penggunaan kabel relatif hemat

## Kekurangan
- Jika jalur utama terganggu, jaringan dapat bermasalah
- Kurang efisien untuk jaringan besar
